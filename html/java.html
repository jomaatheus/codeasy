Introdução à Programação em Java

Java é uma linguagem de programação de propósito geral, amplamente utilizada em diversos tipos de aplicativos, desde aplicações de desktop até aplicações web e móveis. Neste artigo, vamos explorar os conceitos fundamentais da linguagem Java e fornecer exemplos práticos para ajudar a compreender esses conceitos.

Vamos analisar este código:

  • public class HelloWorld: Esta linha define uma classe chamada HelloWorld. Em Java, todo o código deve estar dentro de uma classe.

  • public static void main(String[] args): Este é o método principal (main) que é executado quando o programa é iniciado. Ele é onde a execução do programa começa. public indica que este método pode ser acessado de fora da classe. static indica que o método pertence à classe em vez de instâncias individuais. void significa que o método não retorna nenhum valor. String[] args é uma matriz de argumentos passados para o programa.

  • System.out.println("Hello, World!");: Esta linha imprime "Hello, World!" no console. System.out é um objeto que representa a saída padrão, e println é um método para imprimir uma linha de texto.

Variáveis e Tipos de Dados

Java é uma linguagem fortemente tipada, o que significa que todas as variáveis devem ter um tipo de dados especificado. Abaixo está um exemplo de declaração de variáveis em Java:

System.out.println("Nome: " + nome); System.out.println("Idade: " + idade); System.out.println("Altura: " + altura + "m");

Neste exemplo, declaramos três variáveis: idade do tipo int (inteiro), altura do tipo double (número de ponto flutuante) e nome do tipo String (texto). Em seguida, imprimimos essas variáveis no console.
